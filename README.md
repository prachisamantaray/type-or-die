# 🧟 Type or Die

A fast-paced typing speed game where you must type 
quickly to survive! Choose your chase scenario and 
prove your typing skills.

## 🎮 Live Demo
🔗 [Play Now!](https://your-vercel-link.vercel.app)

## 🎯 Themes

| Theme | Story |
|-------|-------|
| 🧟 Zombie Chase | Type fast or become undead! |
| 💎 Heist Escape | Outrun the police after stealing diamonds! |
| 🚆 Catch the Train | Sprint to catch the last train! |

## ⚡ Features
- 🎨 Animated SVG cartoon characters
- 🔊 Sound effects (footsteps, sirens, groans!)
- 📊 Live WPM & accuracy tracker
- 🏆 Performance grading system (S/A/B/C)
- 🔄 8 unique paragraphs per theme — no repeats!
- 📈 Session stats across multiple rounds
- 3 difficulty levels (Easy / Medium / Hard)
- 🌙 Dark cyberpunk aesthetic
- 📱 Responsive design

## 🕹️ How to Play
1. Choose your **theme** (Zombie / Police / Train)
2. Select **difficulty** (Easy / Medium / Hard)
3. Click **Start** and type the words shown
4. Type **fast and accurately** to escape!
5. Press **Space** after each word to move forward
6. Beat your score each round!

## 🚀 Run Locally
```bash
# Clone the repo
git clone https://github.com/YOURUSERNAME/type-or-die.git

# Go into the folder
cd type-or-die

# Install dependencies
npm install

# Start the game
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) 🎮
